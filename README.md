# Invoice Generator App

an application for creating and managing invoices

## How to Run the App

- Visit https://invoice-generator-app-hng.vercel.app/

## Technologies Used

- Frontend
    - HTML
    - CSS
    - React

- Deployment
    - Vercel (frontend)

## Key Features Implemented

* Create invoices
* Read (view) invoices
* Update invoices
* Save drafts
* Mark invoices as paid
* Filter by invoice status
* Toggle light/dark mode
